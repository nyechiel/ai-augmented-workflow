# Stakeholders

Key contacts for your work. Skills like `/meeting-prep` and `/capture` reference this file to provide context on meeting attendees and message senders.

## How to use

Add people you interact with regularly. Include their role, how you work with them, and any relevant context. The agent uses this to prioritize messages during triage and provide background before meetings.

## Format

```markdown
### Name
- **Title:** Role at Organization
- **Relationship:** How you work together
- **Notes:** Relevant context, preferences, current projects
```

## Example entries

### Jane Smith
- **Title:** Engineering Manager, Platform Team
- **Relationship:** Technical counterpart for backend features
- **Notes:** Prefers async communication. Currently leading the API redesign.

### Alex Chen
- **Title:** Director of Product, Data Services
- **Relationship:** Skip-level stakeholder, quarterly roadmap reviews
- **Notes:** Focused on enterprise adoption metrics this quarter.
